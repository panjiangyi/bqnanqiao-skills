# bqnanqiao-perspective

基于484条推文（2024-2026）的深度分析，蒸馏 [@BQnanqiao](https://x.com/BQnanqiao) 的内容策展思维与编辑眼光。

由 [女娲 · Skill造人术](https://github.com/alchaincyf/nuwa-skill) 生成。

## 这是什么

从一个高互动量的中文内容策展 X/Twitter 账号中提取的**思维操作系统**，包含：

- **5个核心心智模型** — 选题和叙事的编辑视角
- **9条决策启发式** — 什么样的内容能引发共鸣
- **完整的表达DNA** — 句式、语气、节奏、幽默风格

## 核心发现

bqnanqiao 是**内容策展人，不是原创思考者**。97.5%的内容搬运自知乎、抖音、小红书等平台。价值在于编辑品味：选什么故事、怎么讲、什么时候闭嘴。

## 心智模型一览

| 模型 | 一句话 |
|------|--------|
| 故事即武器 | 不要讲道理，讲一个让道理不言自明的故事 |
| 利益透视镜 | 所有看似道德/情感的现象，拉到底看利益流动 |
| 反转即洞察 | 真相在转折处，意外结局揭示一切 |
| 人间两极 | 最好的内容永远在极端——极善或极恶 |
| 不说比说更有力 | 最好的编辑评论是不评论，选题就是态度 |

## 使用方式

将 `SKILL.md` 加载到任何支持自定义提示词/Skill的 AI 助手中。当你需要 bqnanqiao 的视角分析任何问题时，Skill 会自动激活。

## 诚实边界

- 这是一个编辑框架，不是哲学体系
- 账号持有者的真实个人观点完全未知
- 部分内容偏向性别本质主义视角——不确定是个人立场还是流量策略
- 无法预测策展领域之外的原创判断
