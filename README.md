# bqnanqiao-perspective

A distilled skill capturing the editorial thinking and content curation framework of [@BQnanqiao](https://x.com/BQnanqiao), based on analysis of 484 tweets (2024-2026).

Generated by [Nuwa Skill](https://github.com/alchaincyf/nuwa-skill).

## What This Is

This is a **thinking OS** extracted from a high-engagement Chinese content curation account on X/Twitter. It captures:

- **5 Core Mental Models** — the editorial lens for selecting and framing stories
- **9 Decision Heuristics** — rules of thumb for content that resonates
- **Complete Expression DNA** — sentence patterns, tone, rhythm, humor style

## Key Finding

bqnanqiao is a **content curator, not an original thinker**. 97.5% of content is curated from Zhihu, Douyin, Xiaohongshu etc. The value is in the editorial taste: what stories to pick, how to frame them, and when to shut up.

## Mental Models at a Glance

| Model | One-liner |
|-------|-----------|
| Story is Weapon | Don't argue. Tell a story that makes the argument unnecessary. |
| Interest Lens | Behind every moral/emotional claim, trace the flow of interests. |
| Reversal = Insight | The truth is at the twist. The unexpected ending reveals everything. |
| Human Extremes | The best content lives at the extremes — extremely kind or extremely cruel. |
| Silence is Power | The best editorial comment is no comment. Selection IS the statement. |

## Usage

Load `SKILL.md` into any AI assistant that supports custom prompts/skills. The skill activates when you ask for bqnanqiao's perspective on any topic.

## Honesty Boundaries

- This is an editorial framework, not a philosophical system
- The account holder's true personal opinions are unknown
- Some content trends toward gender-essentialist views — unclear if personal belief or engagement strategy
- Cannot predict original judgments on topics outside the curated domain
